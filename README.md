# HTML-CSS-LiveCanvas
**Interactive HTML & CSS Learning Platform**

A modern, interactive educational sandbox designed for Web Development Pedagogy, perfectly blending SEO-friendly concepts, real-time live preview, and educational principles.

## 📋 專案簡介 | Overview
**HTML-CSS-LiveCanvas** 是一套創新的 Web 前端開發教學與沙盒工具，旨在解決傳統教學中「代碼與預覽脫節」的痛點。透過高度互動的實驗室環境，學生可以「即時寫、即時看」 (Live)，無縫修改 HTML 與 CSS 並同步觀察 UI 變化，深度理解網頁結構與樣式間的協同運作。同時，本專案名稱與架構對搜尋引擎最佳化 (SEO) 極度友善，讓初學者能更直覺地連結所學知識與實際應用。

**HTML-CSS-LiveCanvas** is an innovative educational tool (Canvas) designed to bridge the gap between source code and visual output. It provides a real-time (Live), interactive playground where learners can manipulate HTML/CSS parameters and observe UI changes instantaneously, fostering a deep understanding of web development pedagogy. 

## ✨ 核心價值 | Core Values
1. **即時渲染預覽 (Live WYSIWYG)**: 所見即所得，撰寫代碼的當下立刻呈現結果，降低學習挫折感。
2. **教育實驗沙盒 (Pedagogical Canvas)**: 提供無壓力、可重置的畫布空間，鼓勵學生從錯誤與實驗中學習。
3. **SEO 友善架構 (SEO-Optimized)**: 程式碼輸出與語意化標籤的設計，符合現代搜尋引擎規範，將實務與教學結合。

## 🌟 特色功能 | Key Features
- **即時互動教學實驗室 (Live Playground)**
  中：內建拖曳式視窗代碼編輯器，支援 HTML 與 CSS 獨立即時預覽。
  En: Integrated draggable editors with real-time preview, allowing independent manipulation of HTML and CSS.
- **多合一教學模式切換 (Multimodal Teaching Interface)**
  中：提供四種視覺主題 (深色、太陽、藍色、自然)，適應不同教學場景與視覺生理需求。
  En: Features multiple UI themes to optimize for different teaching environments and accessibility requirements.
- **動態參數調教系統 (Dynamic Parametric Controls)**
  中：整合滑桿 (Range inputs) 與單位選取器，直觀展示間距、圓角與陰影的數值變化。
  En: Integrated range sliders and unit selectors to visually demonstrate the impact of altering CSS variables.
- **精準代碼格式化 (Advanced Code Fidelity)**
  中：採用 Prism.js 實現精準的語法高亮，提供媲美 IDE 的閱讀體驗。
  En: Utilizes Prism.js for high-fidelity syntax highlighting.

## 🎓 教育應用價值 | Pedagogical Value
在大學 Web 授課或初學者自學中，此專案可作為：
- **即時演示工具 (Live Demonstration Platform)**：講師可即刻展示 Flexbox 或 Grid 的佈局變化。
- **輕量級作業環境 (Lightweight Lab Environment)**：無需安裝任何軟體，直接透過瀏覽器完成樣式實驗。
- **SEO 與 UI/UX 實踐橋樑 (Bridging SEO & UX)**：整合現代設計思考與搜尋引擎最佳化的實務練習。

## 🛠️ 技術組件 | Technology Stack
- **Core**: Pure HTML5 (Semantic & SEO Friendly), Vanilla JavaScript (ES6+), Modern CSS3 (Custom Variables)
- **Library**: Prism.js for Syntax Highlighting
- **UI Architecture**: Flexible Split-pane & Floating Draggable Layers

